# pulse-Tap

The game should definitely be enjoyable and accessible. It should be the kind of game that you feel compelled to play over and over. This game is very straightforward since it involves just a single button. The game is played through precision and reaction.

This is the action that can be done in this one-button arcade game. The one-button arcade game revolves around skillful execution of the jump action.
